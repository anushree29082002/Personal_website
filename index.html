DigiVoter

DigiVoter is an innovative online voting system platform developed specifically for colleges, enabling seamless online elections for student council members. This advanced system is not limited to college campuses; it can be successfully implemented for elections in any location worldwide. DigiVoter empowers voters to exercise their right to vote conveniently, allowing them to cast their ballots from any location and at any time on a designated day.


Tech Stack

FrontEnd: HTML, CSS, JavaScript

BackEnd: Java, JSP

NewsIn

NewsIN is a remarkable web application that serves as a news summarizer and analyzer for time-conscious individuals. It provides concise summaries, enabling quick grasp of essential information. The analyzer component offers graphical representations, revealing readership patterns and percentage of people reading specific categories. Users can stay informed, save time, and gain deeper understanding of the news landscape with NewsIN.


Tech Stack

FrontEnd: HTML, CSS, JavaScript, Plotly

BackEnd: Python, Streamlit

Database: MySQL

Chefly

Chefly is an innovative web application that generates new and unique recipes using a given list of ingredients. Users can conveniently select ingredients by either entering them as text or uploading an image. The application then generates a customized recipe complete with detailed instructions and the necessary quantity of each ingredient.With Chefly, users can effortlessly explore culinary possibilities and create delicious meals with ease.


Tech Stack

FrontEnd: HTML, CSS, JavaScript

BackEnd: Python, Streamlit

Algorithms:Machine Learning, Deep Learning
